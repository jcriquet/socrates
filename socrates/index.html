Welcome to Web Programming — Spring 2013

Hi. I am Jay Peretz, an Adjunct Faculty member in Computer Studies at Santa Rosa Junior College. I've been teaching web development classes at SRJC since 2009, including subjects such as HTML5, CSS, Javascript, PHP, SQL, XML, Adobe Dreamweaver and Adobe Flash.

This semester I will be teaching CS 50.12 (HTML5) as a online class and CS 53.11b (Advanced Dreamweaver) in Petaluma on Wednesday evenings.

Both classes focus on current HTML5 technologies and students will learn techniques used to develop sophisticated web applications and create information-rich, interactive websites.
